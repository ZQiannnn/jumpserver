----

Jumpserver-Ansible 在 Jumpserver 的基础上添加了在管理资产上执行Ansible Role的功能，无侵入与JumpServer集成，目前适配到0.5.0版本，
如需更新请联系作者。

----

### 功能
  - Ansible Galaxy下载
  - Ansible Role本地上传
  - Playbook支持Tags执行
  - Playbook支持选择系统用户执行
  - Playbook支持绑定资产执行
  - Playbook支持资产的变量绑定
  - Playbook支持WebHook执行
  - Playbook支持查看错误消息
  - 支持局域网的资产管理

### 开始使用

提供了一键安装shell脚本[Docker安装]，脚本参见: utils/jumpserver-install.sh (需要root权限)
```
./jumpserver-install.sh
```
